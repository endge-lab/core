Configuration Context
Сама сущность слоя (то, что у вас сейчас tenant/project/page на разных уровнях).

Context Hierarchy
Иерархия контекстов (Tenant Context -> Project Context -> Page Context).

Facets
Аспекты контекста:
Presentation Facet (UI) и Behavior Facet (контракты/реакции).

Override Cascade
Механизм каскадного переопределения сверху вниз (specific override).

Specificity Order
Порядок приоритета переопределений (кто кого перекрывает).

Effective Configuration
Итоговая «собранная» конфигурация после применения всей каскадной цепочки.

Итоговое название всей модели:
Hierarchical Faceted Configuration Cascade.
